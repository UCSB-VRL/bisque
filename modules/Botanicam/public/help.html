About:

Botanicam 4: The Plant Recognizer module identifies bushes in images and searches for similar images in the users image collection that match the tags for the class of the identified bush. In order to predict the type of bush for the image, the module downloads the latest model file created by the Botanicam Trainer stored on the Bisque system. The feature used to classify is the Homogeneous Texture Descriptor extracted from non-overlapping tiled images of the original image. Each tile is 64 by 64 pixels. The resulting features are classified with Principle Component Analysis and Support Vector Machine. The majority vote is used to decide the class for the image. The confidence in the results section is the amount of features for the majority over the total amount of features. The model files can be found here.

What's new?

Botanicam has moved to use all Python! The new module now uses scikit-learn's SVM module to classify and bisque's feature service for the feature calculations and bisque's very own API to make requests. Check out the source code here!

How to Use:

1. In the "Select Data for Processing" section there are 3 options:

  1. "Select an Image" allows the user to pick an image from their image collection on the Bisque system.
  2. "Select an example" allows the user to test out the module with images that the developer has chosen which would work best with the module.
  3. "Upload local images" which allows the user to select an image from their desktop or other devices to use in the module.

2. Once you have selected an image it will appear in the viewer. A few option will be displayed above the image. If you would like to crop the image select Rectangle and click on the image. A red box will appear on the image. Click the select option and click on the box to drag the box to the region you want to crop. Select the corners and drag to re-size the region.


Figure 1 : Image of a Gobject with the rectangle tool used to crop a section of the image.

3. Click Run and the module will start processing the image. This will take a few minutes.

4. After the modules is done, a detection summary will be given with a list of tags that the module thinks the image should be labelled. The module will also query images on the system with the same tags in the similar images box.
