Process any image/dataset with a CellProfiler pipeline.

  • Note that certain CellProfiler pipeline modules (e.g., 'Images', 'StoreImages', 'ExportToSpreadsheet') are not allowed and need to be replaced with BisQue-specific modules (e.g., 'BisQueLoadImages', 'BisQueSaveImages').
  • Certain other CellProfiler pipeline modules (e.g., 'Crop') are not compatible with BisQue and will be ignored. Image service offers analogous functionality and should be used for any pre-processing.
