About:

Dream3D_UCSB Module runs an arbitrary pipeline on an input HDF resource and generates an output HDF resource.

This module version uses the UCSB branch of Dream3D which may contain additional/different functionality.

How to Use:

1. Select a HDF resource to use for Dream3D.

2. Select a pipeline to use for Dream3D.

3. Name the analysis.

4. Click Run. The module can take from a few minutes to a few hours depending on the pipeline and data size.
