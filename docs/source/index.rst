Welcome to the Bisque Project
=============================


Bisque is an advanced image database and analysis system for
biological 5D images. It supports large scale image databases (TB ->
PB), flexible experimental data management, metatdata and content
based search, analysis integration and knowledge discovery. The system
is based on scalable web services and is deployed on one to N servers
depending on needs and dataload.

Some key ingredients are:
   * Flexible schema for scientific data management
   * Analysis system
   * Search by content and metadata
   * Data visualization system
